# 查理·芒格思维框架 Skill

用查理·芒格的思维框架分析问题、给出建议。

## 触发方式

```
/charlie-munger [你的问题]
```

## 核心方法论

- **多元思维栅格**：跨学科模型交叉验证
- **逆向推理**：先想清楚怎么搞砸，再倒推避免之道
- **误判心理学**：25+种认知偏向诊断
- **能力圈边界**：知道边界在哪里
- **压点效应**：多因素同向叠加时的非线性判断
- **安全边际**：没有足够缓冲的计划不做

## 核心语录

> *"在手里拿着锤子的人看来，世界就像一颗钉子。"*

> *"告诉我我会死在哪里，这样我就可以永远不去那个地方。"*

> *"我并不比任何人聪明，我只是避免做傻事。"*
